# Fanni & Levi -- Esküvői oldala

Egy modern, reszponzív esküvői weboldal HTML5, CSS3 és natív JavaScript-tel.

## Projekt áttekintése

Ez a projekt teljesíti az összes tanulmányi követelményt:
- 6 oldal (4-8 követelmény)
- Teljes reszponzív design (Flexbox, Grid, Media queries)
- Form validálás (7+ mező, 6+ validáció)
- Interaktív fotóalbum (Lightbox galería)
- Natív JavaScript (nincs framework)
- Google Web Font (Playfair Display, Poppins)
- Favicon + metaadatok
- Szervezett fájlstruktúra

## Fájlok

- web.html - Főoldal
- szállás.html - Szállási információk
- schedule.html - Menetrend
- gallery.html - Fotógaléria
- visszajelzés.html - Feedback form
- contact.html - Kapcsolat
- kepek/ - Képek mappája
- others/ - Egyéb fájlok mappája
  - des.css - Stílusok
  - scripts.js - JavaScript
  - favicon.svg - Favikon
  - README.md - Dokumentáció
  - OLVASSEL.txt - Részletes infók

## Funkciók

1. Mobilbarát hamburger menü
2. Interaktív fotóalbum lightbox-szal
3. Form validálás 6 mezőre
4. Teljes reszponzív design
5. Google Web Fonts

## Futtatás

python -m http.server 8000

Majd nyisd meg: http://localhost:8000/web.html
